Epidemic Models

Este projeto tem o objetivo de divulgar como desenvolver modelos para epidemias, desde sua modelagem matemática até sua concepção computacional em Python, sendo composto por vídeos explicativos, Jupyter notebooks e diversas visualizações, para ajudar com o entendimento do conteúdo apresentado.

Documentação

Crie seu próprio modelo

Para que todos e todas consigam fazer uso das técnicas elaboradas no projeto, desenvolvemos um mecanismo para permitir que você também crie um modelo baseado na nossa análise com os dados que você possui.

Para isso, basta enviar um arquivo CSV com seus dados, no formato especificado na tabela abaixo, separados por vírgula. Também é necessário que o arquivo tenha no mínimo 8 registros para que o modelo comece a aprender corretamente.

Tabela de exemplo
data mortes confirmados recuperados novos_confirmados habitantes
0 2020-01-27 0.0 4.0 0.0 4.0 20000
1 2020-01-28 0.0 4.0 2.0 0.0 20000
2 2020-01-29 1.0 4.0 0.0 0.0 20000
3 2020-01-30 0.0 5.0 1.0 1.0 20000
4 2020-01-31 2.0 8.0 1.0 3.0 20000

Tendo essas informações em mãos, aplicaremos as técnicas que desenvolvemos e retornaremos tudo isso pra você em um Jupyter notebook que terá todos os gráficos e códigosVocê pode optar por receber ambos ou apenas os gráficos que geramos, para que você possa fazer suas próprias análises e contribuir no modelamento da pandemia que estiver estudando.
